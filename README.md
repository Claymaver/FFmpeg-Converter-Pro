# FFmpeg Converter Pro

FFmpeg Converter Pro is an easy-to-use, Electron-based video converter with automatic FFmpeg installation. It supports converting video and audio files in multiple formats, with batch processing and quick presets for common media types.

## Features:
- **Video & Audio Conversion**: Support for H.264, H.265, VP9, AAC, MP3, and more.
- **Batch Processing**: Convert entire folders or select files.
- **Quick Presets**: TV Shows, Movies, High Quality, Small Size.
- **Modern UI**: Simple, real-time progress tracking.
- **Auto-Install**: FFmpeg installs automatically on first run.

## Getting Started:

You can find the latest releases [here](https://github.com/Claymaver/FFmpeg-Converter-Pro/releases)

### Requirements:
Node.js 16+
Windows 10/11, macOS 10.13+, or Linux

1. **Clone the repo**:
   ```bash
   git clone https://github.com/Claymaver/FFmpeg-Converter-Pro.git
   cd FFmpeg-Converter-Pro
   ```
2. **install dependencies**:
   ```bash
   npm install
   ```
3. **Run the app**:
   ```bash
   npm start
   ```
4. **Build**(optional):
   ```
   npm run dist
   ```
